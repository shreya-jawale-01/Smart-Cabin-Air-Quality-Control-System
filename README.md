Smart Cabin Air Quality Control System with Automatic Recirculation

This project monitors and controls air quality inside a vehicle cabin using Arduino Uno and MQ-135 sensors. Differential logic is used to compare inside and outside air quality and automatically switch between fresh air, recirculation, and optimal airflow modes.

Components Used
 Arduino Uno
 MQ-135 Gas Sensor -2
 SG90 Servo Motor
 Breadboard
 Jumper Wires

Working
 Fresh Air Mode → Servo at 0°
 Optimal Airflow Mode → Servo at 90°
 Recirculation Mode → Servo at 180°

Features
 Automatic air quality monitoring
 Differential comparison logic
 Automatic flap control using servo motor
